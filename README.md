# Portfolio

This is a modern portfolio website template. It is built using NextJs, Tailwind CSS, ThreeJS & MongoDB. It is fully responsive and can be used for personal portfolio websites.

## Features

- Modern design
- Fully responsive
- Dark mode
- ThreeJS animations
- Contact form
- MongoDB integration
